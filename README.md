En este repositorio yace un pequeño juego de ingenio, escrito en Rust, 
en el cual el jugador es desafiado por un Oráculo silencioso que ha elegido un número secreto. 
Dicho número se encuentra oculto entre el uno y el cien, y solo mediante intentos sucesivos podrá ser revelado.

El propósito de este proyecto es simple: ofrecer un ejercicio lúdico y claro, tanto para quien juega como para quien estudia el código que le da vida.
Herramientas empleadas

Este artefacto fue forjado utilizando:

El lenguaje Rust

La biblioteca eframe junto a egui para la interfaz gráfica

La biblioteca rand para la generación del número secreto
